Quadratische Funktionen – Zusammenfassung

Allgemeine Form: f(x) = ax² + bx + c  (a ≠ 0)

Scheitelpunktform: f(x) = a(x − d)² + e, Scheitel S(d | e)

  • a > 0: Parabel nach oben geöffnet, a < 0: nach unten
  • |a| > 1: gestreckt, |a| < 1: gestaucht
  • Nullstellen über Mitternachtsformel: x = (−b ± √(b² − 4ac)) / 2a
  • Diskriminante D = b² − 4ac entscheidet über Anzahl der Nullstellen

Beispiel: f(x) = 2x² − 4x + 1

⇒ f(x) = 2(x − 1)² − 1, Scheitel S(1 | −1), D = 8 > 0 ⇒ zwei Nullstellen
